# Meetings API

API desenvolvida para eventos beneficentes e encontros religiosos

## Instalação

Use o package manager [npm](https://www.npmjs.com/) para instalar.

```bash
npm install
```

## Desenvolvendo

Use o package manager [npm](https://www.npmjs.com/) para executar a API.

```bash
npm start
```
